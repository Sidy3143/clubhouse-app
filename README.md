# clubhouse-app
 A full-stack members-only message board built with Node.js, Express, PostgreSQL, EJS, and Passport authentication.

Users can create accounts, log in, create posts, become members, and interact with a private community. Non-members can still read posts, but authors remain anonymous unless the user has membership access.

## Features

- User authentication with Passport.js
- Secure password hashing
- Create, edit, and delete posts
- Membership system
- Admin privileges
- Anonymous public posting for non-members
- Server-side rendering with EJS
- PostgreSQL database integration
- Responsive UI with custom CSS
